# 📝 Grievance Management System

A modern **ASP.NET Core MVC** web application that allows users to register, log in securely, and manage grievances through complete **CRUD (Create, Read, Update, Delete)** operations. The project follows the **Repository Design Pattern** to maintain a clean, scalable, and maintainable architecture.

---

## ✨ Features

* 👤 User Registration
* 🔐 Secure User Login
* 📂 Session-Based Authentication
* ➕ Create New Grievances
* 📋 View All Grievances
* ✏️ Update Existing Grievances
* 🗑️ Delete Grievances
* ✅ Server-Side Validation
* 🎨 Responsive UI with Bootstrap
* 🏛️ MVC Architecture
* 📦 Repository Pattern Implementation

---

## 🛠️ Tech Stack

* **Framework:** ASP.NET Core MVC
* **Language:** C#
* **Database:** SQL Server
* **ORM:** Entity Framework Core
* **Frontend:** HTML, CSS, Bootstrap, Razor Views
* **Architecture:** MVC + Repository Pattern

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/grievance-management.git
```

### 2. Open the Project

Open the solution in **Visual Studio 2022**.

### 3. Configure the Database

Update your SQL Server connection string in:

```text
appsettings.json
```

### 4. Apply the Database

Run the project and ensure your database is configured.

### 5. Start the Application

Press **F5** or click **Run** in Visual Studio.

---

## 🔐 Authentication Flow

1. Register a new account.
2. Log in using your credentials.
3. Access the Grievance Dashboard.
4. Perform Create, Read, Update, and Delete operations.
5. Unauthorized users cannot access protected pages.
